Features

Why RickAgent?

Everything you need to run a multi-agent AI system on Telegram, out of the box.

⚡

Zero-Registration Agents

Drop a .py file into agents/ — auto-discovered on next run. No registry, no config file, no restart needed.

🧠

LangGraph Orchestration

Supervisor agent analyzes each request and picks the best team of specialist agents. Multi-agent routing, not single-agent guessing.

🔌

Any OpenAI-Compatible API

Bynara, OpenAI, Groq, Together, Anthropic, OpenRouter, Google Gemini, or any custom endpoint. Switch providers at runtime.

🌐

Bilingual Web Panel

Full RTL Persian dashboard with Vazirmatn font, live SSE activity feed, per-agent config, provider management, and conversation history.

💬

Group Collaboration Mode

Bots collaborate in Telegram groups via cascading replies. Mention a specific bot to route directly, or let all bots collaborate in turn-order.

🛡️

Smart Rate Limiting

Detects 429 errors, tells users exactly when limits reset. Per-chat cooldowns with configurable intervals.

🔧

DB-First Configuration

Zero secrets in .env. All API keys, tokens, and agent configs live in the admin DB — managed entirely from the web panel.

🐍

Python 3.9+ Compatible

No bleeding-edge syntax required. Works on almost any server, any OS. Simple pip install setup.

Architecture

How It Works

Each message goes through intelligent routing to the best team of agents.

1

💬 User Message

User sends a message to the Telegram bot (private chat or group)

↓
2

🧠 Supervisor Analyzes

LangGraph StateGraph routes the message to the optimal combination of specialist agents

↓
3

🤖 Agent Team Executes

Writer, Analyst, Researcher, Planner, and Critic work in parallel or sequence as needed

↓
4

✨ Supervisor Synthesizes

All agent outputs are combined into a single, coherent response

↓
5

📤 Response Delivered

Final response sent back to the user in Telegram

Quick Start

Up & Running in 3 Steps

From clone to running bot in under 2 minutes.

1

Clone the repository

git clone https://github.com/m4tinbeigi-official/RickAgent.git
cd RickAgent
2

Install dependencies

pip install -r requirements.txt

Installs python-telegram-bot, LangGraph, FastAPI, and all dependencies.

3

Start the bot + web panel

python3 main.py

Bot starts polling. Web panel at http://localhost:8080 — configure API keys and bot tokens from there.

🤖 Bot Commands

/start Welcome + team introduction
/agents List active agents with models and run counts
/stats System stats — uptime, token usage, top agents
/panel Web panel link
/setprovider Browse and switch AI providers at runtime
/clear Clear conversation history for this chat
/status Current system configuration and privacy check
