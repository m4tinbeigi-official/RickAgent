Features

Why RickAgent?

Everything you need to run a multi-agent AI system on Telegram, out of the box.

⚡

Zero-Registration Agents

Drop a .py file into agents/ — auto-discovered on next run. No registry, no config file, no restart needed.

🧠

LangGraph Orchestration

Supervisor agent analyzes each request and picks the best team of specialist agents to handle it. Multi-agent routing, not single-agent guessing.

🔌

Any OpenAI-Compatible API

Bynara, OpenAI, Groq, Together, Anthropic, OpenRouter, Google Gemini, or any custom endpoint. Switch providers at runtime with /setprovider.

🌐

Persian Web Panel

RTL dashboard with Vazirmatn font, live SSE activity feed, agent configuration, provider management, and conversation history.

🛡️

Smart Rate Limiting

Detects 429 errors, tells users exactly when limits reset. Per-chat cooldowns with configurable intervals.

📊

Token Usage Tracking

Per-agent input/output token counts, run statistics, uptime tracking — all accessible via commands and the web panel.

🔧

Dynamic Configuration

Override System Prompt, temperature, and model per-agent from the admin panel. No code change needed — hot reload applies instantly.

🐍

Python 3.9+

No bleeding-edge syntax required. Works on almost any server, any OS. Simple pip install setup.

Architecture

How It Works

Each message goes through intelligent routing to the best team of agents.

1

💬 User Message

User sends a message to the Telegram bot

↓
2

🧠 Supervisor Analyzes

LangGraph StateGraph routes the message to the optimal combination of specialist agents

↓
3

🤖 Agent Team Executes

Writer, Analyst, Researcher, Planner, and Critic work in parallel or sequence as needed

↓
4

✨ Supervisor Synthesizes

All agent outputs are combined into a single, coherent response

↓
5

📤 Response Delivered

Final response sent back to the user in Telegram

Quick Start

Up & Running in 3 Steps

From clone to running bot in under 2 minutes.

1

Clone the repository

git clone https://github.com/m4tinbeigi-official/RickAgent.git
cd RickAgent
2

Run the smart installer

python3 install.py

The installer picks your AI provider, tests your API key live, configures Telegram, and writes .env automatically.

3

Start the bot + web panel

python3 main.py

Bot is ready on Telegram. Web panel at http://localhost:8080

🤖 Bot Commands

/startWelcome + team introduction
/agentsList active agents with models and run counts
/statsSystem stats — uptime, token usage, top agents
/panelWeb panel link
/setproviderBrowse and switch AI providers at runtime
/setprovider 2Switch to provider #2 directly
/clearClear conversation history for this chat
/statusCurrent system configuration
